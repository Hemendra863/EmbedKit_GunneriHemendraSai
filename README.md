# EmbedKit_GunneriHemendraSai

## Description
Implementation of an 8-byte Ring Buffer (Circular Buffer) using C.

## Features
- Buffer Initialization
- Write Operation
- Read Operation
- Full Detection
- Empty Detection
- Count Tracking
- Circular Wrap-around

## Build Command

gcc -Wall -std=c99 ringbuf.c -o ringbuf

## Run

./ringbuf
